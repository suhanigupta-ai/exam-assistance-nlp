📚 Exam Assistance System (NLP Project)

🔍 Overview
This project is an AI-powered Exam Assistance System that generates answers based on marks (1, 2, 3, 5, 10 marks).

 🚀 Features
- Extracts text from PDF notes
- Cleans and processes text
- Uses embeddings for semantic understanding
- FAISS-based search (like mini Google)
- Generates exam-oriented answers

 🛠️ Tech Stack
- Python
- PyPDF2
- Sentence Transformers
- FAISS
- NumPy

 ▶️ How to Run

1. Install dependencies:
    pip install -r requirements.txt

2. Run the project:
    py -3.12 main.py


 📌 Example
Input:
- Question: What is a resource?
- Marks: 5

Output:
- Medium-length exam answer
